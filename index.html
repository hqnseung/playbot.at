플레이봇 정답풀이

이 웹사이트는 플레이봇의 이해가 어려운 학생들을 위해 만든 PlayBot의 월드들의 정답과 풀이를 정리한 공간입니다.
최적의 답지를 제공하는건 아니나, 이해를 위해 사용하기에는 충분한 내용입니다.

절대 이 공간은 학습 악용의 목적으로 만든것이 아니며,
학습에 어려움을 느껴 해설이 필요한 학생을 위해 제작한 것이며
단순 복붙과 같은 악용시에는 절대 악용한 당사자의 책임입니다.

            // function 함수이름() {
	          //
            // }

            // 만약 a()를 입력했을때 move()를 하고 set_color("blue")을 작동하게 하고싶다면

            // set_color("blue")

지수 연산자 (** 또는 ^): 첫 번째 숫자를 두 번째 숫자의 지수로 계산합니다. (예: 2^3 = 8)
